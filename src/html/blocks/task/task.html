Задача

  • Анализ рынка, разработка предложений и рекомендаций
  • Написание ТЗ
  • Разработка дизайна
  • Программирование
  • Видеопроизводство
  • Создание 3D моделей конструкций
  • Техническая поддержка
  • Продвижение
